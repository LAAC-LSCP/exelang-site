## VanDam Daylong Demo

A publicly available daylong recording dataset, packaged as a ready-to-use demo corpus for testing and learning.

{{< note >}}
**Use Case:** "I want to try out tools like ChildProject or VTC on real data without setting up my own corpus"

The VanDam Daylong Demo is the ExELang team's repackaging of Mark VanDam's publicly available daylong HomeBank corpus into a ChildProject-compatible format. It serves as a convenient starting point for researchers who want to explore the ExELang toolchain — running annotations, extracting metrics, or following tutorials — using real long-form recording data without needing their own dataset.
